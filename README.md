# DropoutGuard: AI-Based Student Dropout Prediction System

DropoutGuard is a machine learning project that predicts student dropout risk using academic and mental health factors.  
It helps in early identification of at-risk students, enabling timely intervention and improving educational outcomes.

---

## Features

- Predicts student dropout risk using ML models
- Uses academic + behavioral + mental health indicators
- Data preprocessing and feature engineering
- Correlation analysis to identify key factors
- Easy to extend with real-time dashboards

---

## Tech Stack

- **Python**
- **Scikit-learn**
- **Pandas**
- **NumPy**
- **Matplotlib / Seaborn**
- **Machine Learning**

---

## Project Overview

Student dropout is a major issue in education systems.  
DropoutGuard uses data-driven techniques to:

- Analyze student performance
- Identify risk patterns
- Predict dropout probability

This helps institutions take preventive actions before it’s too late.

---

## How It Works

1. Collect student data (academic + behavioral)
2. Clean and preprocess the dataset
3. Perform feature engineering
4. Train ML model (classification)
5. Predict dropout risk
